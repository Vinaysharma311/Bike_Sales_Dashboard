# Bike Sales Dashboard

## Overview

This project presents an interactive Bike Sales Dashboard developed in Microsoft Excel. The dashboard analyzes customer demographics and purchasing behavior to identify patterns that influence bike sales. It transforms raw sales data into meaningful visual insights, enabling data-driven decision-making.

## Features

- Interactive dashboard with slicers and filters
- Analysis based on customer demographics
- Sales trends and purchasing behavior visualization
- Dynamic Pivot Tables and Pivot Charts
- Easy-to-understand KPIs and graphical representation

## Tools & Techniques

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization

## Key Insights

- Customer age groups and income levels impact bike purchasing decisions.
- Commute distance influences the likelihood of purchasing a bike.
- Sales trends can be analyzed across different demographic segments.
- Interactive filters allow quick exploration of the dataset.
